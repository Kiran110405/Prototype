LITERACY

Data

facts and information collected together for reference or analysis. Examples of data that is found online could be: Search history, Online transactions, Social media posts.

Network

A network is about interconnected devices, such as computers, that communicate and share resources using established instructions for the computers to follow. In short, a group of different devices that are linked together that exchange information and share their resources with one another. (like a social circle of people).

Internet

The internet is a global network that provides a large range of information and communication facilities that are able to be used. This consists of interconnected networksusing standardized communications.

Computer

A computer is a electronic device that is able to store all fo your information and also helps in allowing you to browse the world wide web. Computers can be used for many different things, these include: Communication, entertainment, research, games and data storage.

Software

In simple terms, a software is a set of commands/instructions that are created to tell the computer of what to do. It is made up of entire sets of programs, procedures and routines in line with the computers system. A software is often used to tackl/execute a set of specific tasks.

Browser

A browser is a software used to find, gather and also display all of the content inside of the world wide web. A browser allows you to search for things such as webpages, images, videos aswell as other types of files.
